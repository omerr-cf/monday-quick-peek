# Privacy Policy for Monday Quick Peek

**Last updated:** December 2024

## Overview

Monday Quick Peek is a Chrome extension that lets you preview Monday.com task notes on hover. Your privacy is important to us.

## Data Collection

Monday Quick Peek does **NOT** collect, store, or transmit any personal data to external servers.

## Local Storage Only

The extension stores the following data **locally on your device only**:

- Your Monday.com API key (for authentication with Monday.com)
- Daily usage count (for free tier limits)
- Pro license validation status
- User preferences

**This data never leaves your device** and is not accessible to us or any third party.

## Third-Party Services

The extension communicates with these services only when you use it:

### Monday.com API

- **Purpose:** Fetch task notes and updates for preview
- **When:** Only when you hover over a task
- **Data sent:** Your API key and task ID
- **Required:** Yes, for core functionality

### Gumroad API

- **Purpose:** Validate Pro license keys
- **When:** Only when activating a Pro license
- **Data sent:** License key for validation
- **Required:** Only for Pro users

## What We Don't Do

- ❌ We don't collect personal information
- ❌ We don't track your browsing activity
- ❌ We don't sell or share any data
- ❌ We don't use analytics or tracking tools
- ❌ We don't have access to your Monday.com data

## Your Control

You can:

- Remove the extension at any time
- Clear stored data via Chrome's extension settings
- Revoke your Monday.com API key anytime

## Children's Privacy

This extension is not directed at children under 13 and does not knowingly collect information from children.

## Changes to This Policy

We may update this privacy policy from time to time. Changes will be reflected in the "Last updated" date.

## Contact

If you have questions about this privacy policy, please open an issue on our GitHub repository.
